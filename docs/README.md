# Presentation

Open **[index.html](index.html)** in a browser to view the SDK explainer.

> ⚠️ **Download the whole `docs/` folder, not just `index.html`.**
> The page loads its images and video from the `assets/` folder using relative
> paths. If you open `index.html` on its own (without `assets/` next to it),
> the page will load but **the images and video will be missing**.

> ⚠️ **הורידו את כל תיקיית `docs/`, לא רק את `index.html`.**
> הדף טוען את התמונות והסרטון מתוך תיקיית `assets/`. אם תפתחו את `index.html`
> לבד (בלי תיקיית `assets/` לידו) — הדף ייפתח אבל **התמונות והסרטון לא יוצגו**.

| File | Purpose |
|------|---------|
| `index.html` | Main presentation / SDK landing page |
| `mockups.html` | App UI mockups |
| `assets/` | Images and demo video used by the pages |
